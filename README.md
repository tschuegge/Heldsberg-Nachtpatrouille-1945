# Drehbuch für die Nachtpatrouille 1945 im Heldsberg
Dieses Repository enthält das Drehbuch für die Nachtpatrouille 1945 im Festungsmuseum Heldsberg.
Das Schauspiel findet gemeinsam mit Besuchern statt und dauert rund 3 Stunden.
